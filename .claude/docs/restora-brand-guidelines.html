Logo Construction Rules
Mark: Square red container, single character "R" in Bebas Neue, centered. Minimum size: 24px × 24px.
Wordmark: "RESTORA" in Bebas Neue, letter-spacing 0.08em. Never stretch or compress.
Clear space: Minimum clear space equals the height of the "R" mark on all sides.
Animation: The red mark pulses with a soft glow — 3s ease-in-out infinite. Never replace with static.
Usage Rules

DO &
DON'T

✓ Do
Use Bebas Neue for all display headings and section titles
Keep hero and KDS dark — black backgrounds anchor the experience
Use red sparingly (20% max) — on CTAs, labels, and status indicators only
Animate card border sweeps from scaleX(0) → scaleX(1) on hover
Include the section-label red-line prefix pattern on every section
Use cubic-bezier(0.16,1,0.3,1) for all hover and reveal transitions
Keep body text at font-weight 300 for lightness and readability
Use the animated logo pulse on every header instance
✕ Don't
Use Inter, Roboto, or system-ui for any displayed text
Apply red to decorative or background elements — it dilutes impact
Use three or more consecutive sections with the same background color
Use ease or ease-in-out for transitions — always use the defined curves
Put gradients or textures on the red accent color — keep it flat
Use body text heavier than font-weight 400 — reserve bold for labels only
Resize or distort the logo mark — use SVG scaling only
Show disabled states in red — use muted border and grey text instead
